●cut除去定理
シークエント Γ → Δ がLKで証明可能ならば、
Γ → Δ に至る証明図でcutを一度も用いないものが存在する。

● sub-formula property
Pをシークエント Γ → Δ に至るcutなしの証明図とする。
Pに現れるどの論理式も、シークエント Γ → Δ に現れるどれかの論理式の部分論理式になっている。

sub-formula propertyより、Pに現れるどの論理式も、
Γ' → Δ' に現れる論理式のどれかの部分論理式となっている。

ここで、あるシークエント Γ0 → Δ0 に推論規則を適応して既約なシークエント Γ0' → Δ0' が導かれたとすると、
Γ0' と Δ0'　に属する論理式の部分論理式となり得る論理式の数は有限である。
従って、それらの論理式からなる既約なシークエント全体の集合Wも有限である。

●双対性
¬ に関する推論規則 ... ¬左と¬右の推論規則が、対称な形をしている
∧ と ∨ の推論規則 ...　推論規則の左右、→の左右、∧と∨の、それぞれを入れ替えると似た形になる

●双対定理
AとBを⊃を一つも持たない論理式とし、∧と∨を入れ替えた論理式をそれぞれA'とB'とする。
その時、以下が成り立つ
(1) A → B がLKで証明可能ならば、B' → A' もLKにて証明可能
(2) A ≡ B がLKで証明可能ならば、B' ≡ A' もLKにて証明可能


(1)の証明
cut除去定理及びsub-form propertyより、A → B についてcutと⊃の推論規則を含まない証明図Pが存在する。
Pに現れるシークエントの左辺と右辺、∧と∨を入れ替えると A' → B' の証明図となる


(2)の証明
A ≡ B が証明可能   <==> (A ⊃ B) ∧ (B ⊃ A)　が証明可能
  　　　　　　　　  <==> (A ⊃ B) と (B ⊃ A)　が証明可能
同様に
A' ≡ B' が証明可能 <==> (A' ⊃ B') と (B' ⊃ A')　が証明可能
以上と(1)から、
  A ≡ B が証明可能 <==> A' ≡ B' が証明可能
が分かる
