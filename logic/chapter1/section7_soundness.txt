
●トートロジーな式
トートロジーは論理式に対してしか定義されていない。
　=> 論理式Aがトートロジー iff forall v, v(A) = t

証明可能な式と、トートロジーな論理式との関係を考えるために、
トートロジーの概念をシークエントにも適応するよう拡張する。

Γを論理式の列{A₁, ..., Aₘ}とする時
Γ⁺ := A₁ ∨ ... ∨ Aₘ (m > 0)
      ⊥ (m = 0)
Γ₊ := A₁ ∧ ... ∧ Aₘ (m > 0)
      ⊤ (m = 0)
と定義する

Γ₊ ⊃ Δ⁺ がトートロジーである時、式Γ → Δがトートロジーであると呼ぶ

●健全性
証明可能な式がすべてトートロジーである事を健全性と呼ぶ

●完全性
トートロジーな式がすべて証明可能である事を完全性と呼ぶ

（＊）上記定義において、式を論理式に変えても内容は変わらない。なぜなら、
      Γ → Δ が証明可能　   <==> Γ₊ ⊃ Δ⁺　が証明可能     (例1.15)
      Γ → Δ がトートロジー　<==> Γ₊ ⊃ Δ⁺　がトートロジー (定義)
    　が言えるから


●健全性の証明
Γ → Δ が証明可能ならば Γ₊ ⊃ Δ⁺がトートロジーである事を示す。
Γ → Δに関する構造的帰納法を行うと、真理値を網羅的に試す事で証明できる。
